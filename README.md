# renovate-config

Shared Renovate configuration files for my personal projects.

To use it, create a Renovate configuration file in your repository, for example as `.github/renovate.json`:

```json
{
	"$schema": "https://docs.renovatebot.com/renovate-schema.json",
	"extends": ["github>delucis/renovate-config"]
}
```

[View the config file](./default.json)

[Learn more about Renovate config presets](https://docs.renovatebot.com/config-presets/)
